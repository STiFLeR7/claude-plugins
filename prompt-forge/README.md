# Prompt Forge Plugin

Advanced prompt refinement skill for Claude Code and Gemini CLI. It extracts your real intent, investigates your codebase, and surfaces the perspectives that fatigue makes you forget.

## Installation

### For Claude Code
```bash
claude plugins link D:\prompt-forge-plugin
```

### For Gemini CLI
```bash
gemini extensions link D:\prompt-forge-plugin
```

## Features
- **Intent Extraction:** Turns "fix the bug" into a grounded technical brief.
- **Deep Grounding:** Analyzes your codebase and ecosystem before writing the prompt.
- **Perspective Lenses:** Considers Security, QA, UX, Performance, and 5 other critical angles.
- **Plugin Integration:** Optimizes output for GSD and Superpowers workflows.

## How it Works
1. Trigger the skill with phrases like "help me prompt" or "improve my prompt".
2. The skill investigates your codebase and `CLAUDE.md`.
3. It asks sharp, easy-to-answer clarifying questions.
4. It delivers a refined, grounded prompt for you to copy-paste into your execution tool.
